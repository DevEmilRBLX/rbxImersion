# rbxImmersion
rbxImmersion is a module desinged to improve the immersion in Roblox Games by adding visual effects

## Installation:
- Download the package from the Creator Store [here](https://create.roblox.com/store/asset/123539083892419/rbxImersion)
- Place the Package inside of Replicated Storage or StarterPlayerScripts

Wally installlation might be coming soon

## Use of components
```lua
local ComponentModule = require(path.to.rbxImmersion.script).componentName
local Component = ComponentModule.Get()
Component:Method()
```
